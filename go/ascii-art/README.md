# ASCII Art Generator in Go

This project is part of my **Software Engineering Knowledge Library** and documents everything I learned while building an ASCII Art generator in Go.

Rather than focusing only on the final solution, this series explains the concepts, algorithms, design decisions, and implementation behind the project. The goal is to help readers understand not just **what** the code does, but also **why** it works.
